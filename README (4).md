# SQL_Project_1.xlsx

## Overview

This Excel file is designed for practice with **SQL concepts, data
analysis, and Excel calculations**.\
It contains multiple datasets that can be imported into a database or
used directly in Excel for analysis.

## Sheets Description

### 1. Students_Data

Contains student academic information.

**Columns:** - Student ID -- Unique student identifier\
- Name -- Student name\
- Math -- Math marks\
- Science -- Science marks\
- English -- English marks\
- Enrollment Date -- Date of enrollment\
- Grade -- Overall grade\
- Math & Science \>80 -- Indicates students scoring above 80 in both
subjects

**Use case:**\
- SQL `WHERE`, `AND`, `CASE`, filtering, grading logic

------------------------------------------------------------------------

### 2. Sales_Data

Sales transaction details.

**Columns:** - Sales ID -- Unique sale identifier\
- Product -- Product name\
- Region -- Sales region\
- Salesperson -- Person responsible for sale\
- Amount -- Sales amount\
- Date -- Sales date

**Use case:**\
- SQL `GROUP BY`, `SUM`, `ORDER BY`, region-wise analysis

------------------------------------------------------------------------

### 3. Employees_Data

Employee information dataset.

**Columns:** - Employee ID -- Unique employee identifier\
- Name -- Employee name\
- Department -- Department name\
- Salary -- Monthly salary\
- Joining Date -- Date of joining\
- Experience (Years) -- Total experience

**Use case:**\
- SQL `AVG`, `MAX`, `MIN`, `JOIN`, HR analytics

------------------------------------------------------------------------

### 4. Calculations

Summary and calculated values.

**Contains:** - Total number of students scoring more than 50 in Math

**Use case:**\
- Excel formulas, validation of SQL query results

------------------------------------------------------------------------

## Recommended Usage

-   Import sheets into MySQL / PostgreSQL / SQL Server
-   Practice `SELECT`, `FILTER`, `AGGREGATE`, and `DATE` functions
-   Use Excel formulas to cross-check SQL outputs
